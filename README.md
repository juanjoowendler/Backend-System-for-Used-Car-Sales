# REST API for vehicle operations management

Microservices-based backend for managing road tests, vehicle geolocation, and operational reports.

### Main idea:
Control and ensure the use of vehicles during driving tests through business logic and automated validations.


### Technologies used:
- Java 21
- Spring Boot (Spring Data JPA, Spring Security)
- Hibernate / JPA
- Spring Cloud Gateway
- SQLite
- JWT (authentication)
- Apache POI (export to Excel)
- Testing with Spring Boot Test

  
### Architectural aspects:
- Microservices architecture
- API Gateway as the entry point
- Services decoupled by responsibility
- Role-based authentication and authorization
- Persistence with a relational database
- Integrated automated testing
